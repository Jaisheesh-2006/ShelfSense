# Current State

Project state now lives in the LLM wiki. Start here every session:

- **[docs/wiki/README.md](docs/wiki/README.md)** — session bootstrap / index (read first)
- [docs/wiki/STATE.md](docs/wiki/STATE.md) — current phase + next action
- [docs/wiki/GROUND_TRUTH.md](docs/wiki/GROUND_TRUTH.md) — facts distilled from `docs/raw/`

`docs/raw/` is user-owned source material. `docs/wiki/` is the assistant's synthesized
understanding, updated continuously (Karpathy LLM-wiki pattern).
